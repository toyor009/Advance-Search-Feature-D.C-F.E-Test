# Front-end Test - Building a Search Box Component By Toyosi

This is my implementation of the Front-end Test for D.C, using:

- Vue 3
- TypeScript
- Tailwind CSS

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
